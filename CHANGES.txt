================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

If tests fail:
- Reinstall: pip install --upgrade openai geopy
- Check: python -c "import openai; import geopy; print('OK')"
- Run again: python test_new_extraction.py

If OpenAI extraction doesn't work:
- Check API key in .env (should start with "sk-proj-")
- Verify key from: https://platform.openai.com/api-keys
- Check API usage: https://platform.openai.com/account/usage/overview

If geocoding is slow:
- Normal latency for Nominatim is 1-3 seconds
- Consider caching results for production

If port 8000 is in use:
- Use different port: uvicorn main:app --port 8001
- Or kill existing: lsof -i :8000 && kill -9 <PID>
